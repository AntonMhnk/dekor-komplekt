# Декор Комплект

Сайт-каталог для компании "Декор Комплект" — продажа отделочных материалов.

## Категории товаров

- Краска
- Декоративная штукатурка
- Обои
- Фрески
- Лепной декор
- Пол
- Ковры
- Клеевые материалы

## Технологии

- React 18
- React Router 6
- Vite
- CSS (без препроцессоров, для простоты)

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Структура проекта

```
src/
├── components/     # Компоненты (Header, Footer, ProductCard)
├── pages/          # Страницы (Home, Catalog, Category, Product, Contact)
├── data/           # Данные товаров и категорий
├── styles/         # Глобальные стили
├── App.jsx         # Главный компонент с роутингом
└── main.jsx        # Точка входа
```

## Особенности

- Лаконичный минималистичный дизайн
- Адаптивная верстка (mobile-first)
- Карточки товаров с hover-эффектами
- Страницы категорий с hero-секцией
- Форма обратной связи
